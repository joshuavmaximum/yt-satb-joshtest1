How to Use

  1. Enter a YouTube URL - Paste any valid YouTube video URL
  2. Validate the URL - Click the validate button to check the URL
  3. Choose Settings - Select audio quality and MIDI resolution
  4. Process Video - Click "Process Video" to start conversion
  5. Wait for Processing - Monitor progress in real-time
  6. Download MIDI Files - Download individual tracks as MIDI files

Technical Details

This application uses:

  • Spleeter API - For audio separation into multiple stems
  • Web Audio API - For client-side audio processing
  • MIDI.js - For MIDI file generation and playback
  • Modern JavaScript - ES6+ features for better performance
  • Responsive CSS - Grid and Flexbox for modern layouts
