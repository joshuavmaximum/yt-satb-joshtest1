How it works

1. Extract Audio

Downloads and extracts audio from the YouTube video using yt-dlp

2. Split Audio

Uses Spleeter to separate the audio into 5 stems: vocals, drums, bass, piano, and other

3. Convert to MIDI

Converts each audio stem to MIDI using pitch detection and rhythm analysis

4. Download Files

Download individual MIDI files for each instrument track

Powered by Spleeter API and Web Audio API
